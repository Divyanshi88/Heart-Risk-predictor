About the Project

Heart Risk Prediction System

This project aims to predict the risk of heart disease using advanced machine learning techniques. We've developed an ensemble model that combines multiple algorithms to provide accurate predictions based on various health parameters.

Key Features:
  • Ensemble machine learning model
  • User-friendly interface
  • Detailed risk assessment
  • Performance visualization
